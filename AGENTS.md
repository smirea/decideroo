# Stack

- Tooling: Bun + TypeScript
- Server: Bun.serve API
- Client: React + Vite + TanStack Router
- UI: Tailwind CSS (enabled in client/src/index.css), framer motion for complex animations
- Linting and Hooks: Oxlint + oxfmt + Lefthook

We use localias to run the client dev server on https://decideroo.localhost locally and the api on https://decideroo-api.localhost

I am already running `bun dev` so just assume it's always running and up to date
